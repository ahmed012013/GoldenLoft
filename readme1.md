# 🕊️ GoldenLoft - Pigeon Loft Management System
> **النسخة الفابريكا لإدارة غيات الحمام باحترافية**

---

## 📋 نظرة عامة (Project Overview)
**GoldenLoft** هو نظام متكامل مصمم لمربي الحمام (الجيّار) والمهتمين بالسلالات، عشان يحول الهواية لنظام رقمي دقيق. المشروع بيجمع بين قوة **NestJS** في الباك إند وجمال وسرعة **Next.js** في الفرونت إند، مع دمج الذكاء الاصطناعي لإدارة صحة وتغذية الطيور.

---

## 🛠️ المواصفات التقنية (Tech Stack)

| الجزء | التقنية المستخدمة | الوصف |
| :--- | :--- | :--- |
| **Frontend** | Next.js 14+ | واجهة مستخدم سريعة، متجاوبة (Responsive) وتدعم الـ RTL. |
| **Backend** | NestJS | نظام Modular قوي لإدارة الـ APIs والـ Business Logic. |
| **Database** | PostgreSQL (Neon) | داتابيز سحابية لضمان حفظ البيانات وسرعة الوصول. |
| **ORM** | Prisma | لسهولة التعامل مع قاعدة البيانات وعمل الميجريشن. |
| **AI Engine** | Ollama | لدمج نماذج ذكاء اصطناعي محلية (Llama/Phi) لاستشارات المربين. |
| **DevOps** | AWS EC2 & PM2 | استضافة السيرفر وضمان تشغيله 24/7. |

---

## ✨ المميزات الرئيسية (Core Features)

### 1. إدارة الطيور (Pigeon Management)
* تسجيل بيانات الحمام (النوع، السلالة، تاريخ الميلاد، اللون).
* نظام "شجرة العائلة" لتتبع الأصول والنتائج.

### 2. السجلات الصحية (Health Records)
* تتبع التحصينات، الأمراض، والعلاجات.
* تنبيهات بمواعيد الأدوية الدورية.

### 3. الإدارة المالية (Financial Tracking)
* حساب تكاليف العلف، العلاج، وعمليات البيع والشراء.
* تقارير دورية عن أداء "اللوفت" مالياً.

### 4. المساعد الذكي (AI Assistant)
* دمج **Ollama** للرد على استفسارات المربي بخصوص الأمراض المفاجئة أو طرق التغذية المثالية بناءً على بيانات السلالة.
